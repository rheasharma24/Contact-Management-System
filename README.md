# Contact-Management-System
java project for managing contact
A simple and efficient Contact Management System built using Java.
This project allows users to store, manage, and organize contacts using basic data structures.

# Features
* Add new contacts
* View all contacts
* Search contacts by name
* Delete contacts
* Update contact details

# Technologies Used 
*Java
ArrayList (for dynamic storage)
Object-Oriented Programming (OOP) concepts

This project is created for learning purposes and demonstrates basic Java programming and data structure usage.
